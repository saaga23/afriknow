# Data

The datasets for this project are hosted on Kaggle.

## AfriKnow v18 Inputs

**URL:** https://www.kaggle.com/datasets/abrahamsunday123/afriknow-v18-inputs

**Contents:**
- `03_openrouter_outputs_v18_correct.csv` — 180 items × 7 models, greedy + VCE
- `03_openrouter_manifest_v18.json` — Run manifest
- `README.md` — Dataset documentation
- `credentials.env` — API keys (private)

## How to Download

```bash
# Using Kaggle API
pip install kaggle
kaggle datasets download abrahamsunday123/afriknow-v18-inputs --unzip
```

Or download directly from the Kaggle website.
